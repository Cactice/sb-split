# sb-split
split md file to scrapbox appropriate format

# What is this

It splits md files to an aprpropriate file for scrapbox.

# How to get started
Both backend and frontend is needed to run the app.

backend
```
cd experiments
FLASK_APP=server.py flask run
```

frontend
```
cd frontend
npm i
npm run dev
```
